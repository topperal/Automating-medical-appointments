# Automating-medical-appointments

Система для автоматизации медицинских назначений на основе голосового помощника Яндекс Алиса.

## Общая структура проекта

### Каталог `Research`
Содержит Jupyter-ноутбуки с исследованиями библиотек для решения задач OCR (оптического распознавания символов) и NER (извлечения именованных сущностей). В данном разделе также представлена разработанная методика объективной оценки качества TTS-систем в условиях больничного шума, а также результаты сравнительного анализа двух моделей по этой методике.

### Каталог `Server`
Содержит ноутбук, описывающий все этапы обработки амбулаторной карты пациента на серверной стороне. Внутри каталога находится папка `Application`, включающая все компоненты, необходимые для локального развертывания сервера на компьютере с интуитивно понятным графическим интерфейсом пользователя.
